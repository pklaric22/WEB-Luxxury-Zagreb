# WEB-Luxxury-Zagreb
## Opis

Ovo web sjedište implementirano kao web trgovina za prodaju luksuznih satova bilo je završni rad u Tehničkoj školi Ruđera Boškovića gdje sam stekao zvanje Tehničar za računalstvo.

## Korištene tehnologije
* HTML 5
* CSS
* PHP
* JavaScript
